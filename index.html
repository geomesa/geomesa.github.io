GeoMesa is an open-source, distributed, spatio-temporal database built on top of the Apache Accumulo column family store. Leveraging a highly parallelized indexing strategy, GeoMesa aims to provide as much of the spatial querying and data manipulation to Accumulo as PostGIS does to Postgres.

Quick start

Navigate to where you would like to download the project and run:

git clone git@github.com:locationtech/geomesa.git

GeoMesa is managed by Maven, and is built with the command:

mvn clean install

From the root directory, this builds each sub-project with its additional dependencies-included JAR.

For a more detailed introduction, please read through the tutorials.

Roadmap

GeoMesa aims for a 1.0 release in September. Features that remain to be implemented include:

  • deep integration with Accumulo and Geoserver security
  • query-planning optimizations
  • optimization of storage and querying of lines and polygons

Downloads

GeoMesa artifacts including client API jars and deployable Accumulo extensions will be in Maven Central. For now, users should build the project locally using Maven.

Get involved
