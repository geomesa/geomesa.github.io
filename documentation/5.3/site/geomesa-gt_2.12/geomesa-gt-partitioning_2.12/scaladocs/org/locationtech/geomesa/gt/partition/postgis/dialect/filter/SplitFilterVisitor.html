SplitFilterVisitor 

Companion object SplitFilterVisitor

class SplitFilterVisitor extends DuplicatingFilterVisitor

Filter visitor we use for processing filters in the dialect's splitFiltermethod.

Currently the filter does the following:

  • replaces constant functions with their literal values
  • removes "whole-world" geometry filters
