ReceiptTimeScheme 

Companion object ReceiptTimeScheme

case class ReceiptTimeScheme(delegate: DateTimeScheme, buffer: Duration) extends PartitionScheme with Product with Serializable

Scheme for partitioning based on "receipt time", i.e. when a message is received. Generally this is useful only for reading existing data that may have been aggregated and stored by an external process.

delegate

delegate date time scheme options

buffer

amount of time to buffer queries by, in order to match a feature date to a receipt time date - i.e. the amount of latency in the ingest process
