17. Bigtable Data StoreÂ¶

The GeoMesa Bigtable Data Store is an implementation of the GeoTools DataStore interface that is backed by Google Cloud Bigtable.

The Bigtable code is based on the Bigtable HBase API, and found in the source distribution under geomesa-bigtable/.
