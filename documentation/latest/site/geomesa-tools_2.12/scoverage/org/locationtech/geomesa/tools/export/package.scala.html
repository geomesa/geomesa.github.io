17   /**
18    * Creates unique file names generated from a base name, by appending a sequence number
19    * before the file extension.
20    *
21    * For example, given the file name 'foo.txt', will return 'foo_000.txt', 'foo_001.txt', etc.
22    * If the iterator exceeds the specified number of digits, it will start to append additional
23    * digits to ensure uniqueness, e.g. 'foo_999.txt', 'foo_1000.txt', 'foo_1001.txt', etc.
24    *
25    * @param path file name path
26    * @param digits number of digits used to format the sequence number
27    */
