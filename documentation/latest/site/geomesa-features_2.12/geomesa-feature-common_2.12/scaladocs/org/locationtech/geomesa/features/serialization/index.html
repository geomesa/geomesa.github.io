  2.  class GeoJsonSerializer extends LazyLogging

    Serializer to write out geojson from simple features.

    Serializer to write out geojson from simple features. There are two valid usage patterns.

    To encode a feature collection with an array of features:

    <ol> <li>startFeatureCollection</li> <li>write (0-n)</li> <li>endFeatureCollection</li> </ol>

    To encode individual features:

    <ol> <li>write (0-n)</li> </ol>

    Note that encoding individual features is not valid json, so generally you would want to split up the output after each feature.
